He desarrollado un sitio web como Curriculum Vitae utilizando Angular 18, el uso de componentes reutilizables para evitar la duplicación de datos. Los datos que presenta el sitio están almacenados en un archivo JSON, el cual actúa como base de datos, y son recuperados de manera eficiente a través de servicios inyectables (Inject), garantizando una estructura organizada y limpia.

La página está organizada en tres partes principales: aside, section y nav. En el aside, muestro mis datos personales, mientras que en la section se presenta la información principal de mi Curriculum Vitae. El nav contiene el sistema de routing, que permite navegar de manera eficiente entre las diferentes secciones de la página sin necesidad de recargarla. Además, en esta área he añadido un componente que muestra la fecha y hora actuales en tiempo real, actualizándose continuamente.
